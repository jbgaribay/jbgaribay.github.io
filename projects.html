My Projects

Here are some of the projects I've been working on. Each one represents a learning experience and a step forward in my journey as a developer.


► Networked Snake Game

Description: A real-time multiplayer implementation of the classic Snake game built with a client-server architecture using low-level C systems programming. The server maintains the authoritative game state using a linked-list data structure, while the client handles user input through raw terminal mode. Features include event-driven architecture using select() for I/O multiplexing, non-blocking input handling, and timeout-based game loop ensuring continuous snake movement.

Technologies: C, IPv6 Sockets, TCP/IP, POSIX APIs (termios, fcntl, select), Linked Lists

Key Features: Client-server architecture, non-blocking I/O, raw terminal control, collision detection, dynamic memory management

Status: Completed

Links: GitHub Repo


► Buddhabrot Fractal Generator

Description: A computational art project that generates high-resolution Buddhabrot fractals - a variant of the famous Mandelbrot set that traces the paths of escaping points in the complex plane. The algorithm iterates z_(n+1) = z_n^2 + c for complex values, tracking trajectories that escape to infinity. Different iteration counts for RGB channels create stunning colorful visualizations enhanced through histogram equalization.

Technologies: C, Complex Number Arithmetic, Image Processing, PPM Format

Key Features: 4000x4000 resolution rendering, multi-channel color processing, histogram equalization, coordinate transformation, manual memory management

Status: Completed

Links: GitHub Repo | Sample Images


► Epidemic Spread Simulator

Description: An interactive epidemic modeling tool that simulates disease transmission across social networks using the SEIR (Susceptible-Exposed-Infected-Recovered) epidemiological model. Models disease spread using graph theory where nodes represent individuals and edges represent social connections. Users can adjust transmission rates, vaccination probability, and network topology to observe outbreak dynamics in real-time.

Technologies: Python, NetworkX, Matplotlib, Jupyter Widgets, NumPy

Key Features: SEIR model with vaccination, interactive parameter control, network-based transmission, real-time visualization, configurable topology

Status: Completed

Links: GitHub Repo | Live Notebook


► Shell Sort Gap Sequence Analyzer

Description: A visualization and benchmarking tool that compares different gap sequence strategies for the Shell Sort algorithm. Empirically compares Shell's Original, Ciura's, Sedgewick's, and Tokuda's sequences through animated sorting visualizations and statistical analysis. The tool measures execution time across multiple trials and array sizes to determine optimal gap sequences.

Technologies: Python, NumPy, Tkinter, ttkbootstrap, Pickle

Key Features: Real-time sorting visualization, performance benchmarking, statistical analysis, persistent data storage, interactive GUI, configurable array sizes

Status: Completed

Links: GitHub Repo
